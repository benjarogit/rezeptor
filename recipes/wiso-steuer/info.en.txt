# WISO Steuer (Portable)

Author: benjarogit
Version: WISO Steuer 2026 (Portable 33.05 Build 3220)

## Summary
WISO Steuer as a portable app on Proton-GE — pick a source, copy it to the target folder, then launch.

## Requirements
• Proton-GE (used automatically)
• Portable folder or archive (with start.exe or Steuersoftware 20XX)

## Install
1. Choose Install
2. Pick the source: folder or archive
3. Choose the target folder (default: ~/Documents/WISO Steuer 2026)

## Usage
• Launch via start.exe, or click Launch in Rezeptor
• Optional: Patch.exe and VC_redist as described in the ReadMe

## Notes
• If a CRC32 warning appears, confirm with Yes
• Logs: Settings under the Rezeptor menu
