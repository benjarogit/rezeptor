# Zombie Army 4: Dead War — +13 Trainer (Baracuda)

Author: benjarogit
Version: build 2020.10.21.973201 (Baracuda +13, Update 02.11.2024)

## Summary
Cheat Engine trainer (**Baracuda +13**) for **Zombie Army 4: Dead War**
(Steam AppID 694280). Separate recipe — the Plus 14 / FLiNG trainer is
“Zombie Army 4 — Plus 14 Trainer”.

## Requirements
• Steam and Zombie Army 4: Dead War are installed
• The game has been launched at least once with Proton / GE-Proton
• The Baracuda **.exe** is available locally

## Install
1. Choose Install
2. Select the `… {Baracuda}.EXE` file
3. Check the target folder

## Usage
• Start the game first
• Then click Launch in Rezeptor
• Use hotkeys/options in the trainer window

## Notes
• Logs: ~/.local/share/wine-software/logs/
• Antivirus may block CE trainers — the temp folder must allow execution
